## _*KioskApp - NVR_
A kiosk tablet application use for self check-in in Royal Papua Yacht Club. The app has a facial detection feature that automatically captures the user's face when detected. It is recognized and validated through the AnyVision server using RESTful API. Has the capabililty to scan RFID cards using an external RFID reader for member validation. It also has the capability to print badge upon guest registration.

<a href='https://play.google.com/store/apps/details?id=com.oneberry.kioskappnvr'><img alt='Get it on Google Play' src='https://play.google.com/intl/en_us/badges/images/generic/en_badge_web_generic.png' height='80px'/></a>
